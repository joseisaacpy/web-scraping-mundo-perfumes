# Perfume Web Scraper

Script de Web Scraping desenvolvido em Python utilizando Selenium e Pandas para coletar dados de perfumes (nome e preço) de um e-commerce e exportar os resultados para um arquivo CSV.

## Funcionalidades

- Abre o site automaticamente com Selenium
- Navega por todas as páginas disponíveis
- Extrai nome e preço dos produtos
- Armazena os dados em um arquivo CSV
- Estrutura modular e organizada

## Tecnologias utilizadas

- Python 3
- Selenium
- Pandas
- WebDriver Manager
